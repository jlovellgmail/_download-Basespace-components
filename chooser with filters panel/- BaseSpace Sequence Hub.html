Select All
Select the project that stores the analysis results.
Select All
Browse to the sample you want to analyze and select the check box. You can analyze multiple samples.
Select a panel of targeted amplicons representative of the selected samples for analysis.
Select All
Select a TruSeq Custom Amplicon manifest file to be used for analysis (only available when "Custom Panel" is selected in the "Targeted Amplicons" drop-down list). To upload a new TruSeq Amplicon custom manifest file, navigate to a Project and select the "Import" feature.
Choose which variant caller to use to use. Somatic variant caller is recommended for "TruSeq Amplicon - Cancer Panel," "TruSight Cancer Hotspot Panel," and "TruSight Myeloid Sequencing Panel."
Select the threshold level above which somatic variants will be called. Low threshold values may result in more false positive variants.
Overlapping parts of paired-end reads will be combined to create a single (longer) read for alignment.
Choose which gene and transcript annotation reference database to use.
